# Vision Graphics — Infographic Design Guide

Use this guide when creating infographics for Vision Graphics Kft. in nano banana or any other design tool.
The goal is visual consistency with the website: dark luxury editorial, precise, credible.

---

## Brand Identity

**Company:** Vision Graphics Kft. — Budapest, founded 1996.
**Who they are:** Solo architectural visualization studio run by László Kerezsi.
30 years in 3ds Max, AI integration, Unreal Engine VR, custom scripting.
**Tone:** Technically confident, direct, no fluff. Not a generic agency.
**Tagline space:** Each service has its own tagline — use them as headline anchors.

## Color Palette

### Primary — Dark Mode (preferred for infographics)

| Role | Hex | Use |
|---|---|---|
| Background | `#1a1a1a` | Canvas / slide background |
| Surface | `#121212` | Cards, boxes, panels |
| Surface Raised | `#0f0f0f` | Inset panels, table rows |
| Border | `#333333` | Dividers, outlines, rules |
| **Accent Red** | `#da1313` | **Primary highlight — use sparingly** |
| Text / Headings | `#f0f0f1` | All primary text |
| Text Muted | `#c8c8c8` | Secondary labels, captions |
| Text Faint | `#8a8a8a` | Tertiary info, disabled states |

### Secondary — Light Mode (for print / white-background outputs)

| Role | Hex |
|---|---|
| Background | `#f7f5f2` |
| Surface | `#eeeae5` |
| Border | `#d0cbc3` |
| Accent Red | `#c41010` |
| Text | `#18160f` |
| Text Muted | `#47433b` |

### Usage rules
- Red (`#da1313`) is for emphasis only: step numbers, key figures, callout labels, active states.
- Never use red as a background fill over large areas.
- Prefer near-black surfaces over pure black.
- White is never used — use `#f0f0f1` for the lightest text.

**Font:** Work Sans — exclusively. No other fonts.
**Weight range:** 300 (light captions) → 400 (body) → 600 (labels) → 700 (headings)

## Visual Aesthetic

**Style:** Dark luxury editorial. Think architectural press, not SaaS startup.
- Minimal decorative elements — structure comes from layout, not ornament
- Precision over decoration — clean lines, even spacing, no shadows unless purposeful
- Photography feel — deep blacks, high contrast, deliberate use of red as the only warm note
- No gradients unless subtle (dark-to-darker, not rainbow)
- No rounded corners on containers (2–4px max if any)
- No drop shadows on text

**What this is NOT:**
- No neon / glow effects
- No bubbly icons
- No infographic clip-art
- No stock illustration style
- No serif fonts

---

## Layout Principles

- **Left-aligned** text within containers (centered only for isolated stat callouts)
- **Grid-based** — 8px base grid, all spacing multiples of 8
- **Generous negative space** — breathing room is part of the luxury feel
- **Hierarchy through size and color**, not shape variation
- Containers: thin `1px` border in `#333333`, filled with `#121212`

---

## Iconography

- Minimal line icons — 1.5–2px stroke weight, no fill
- Geometric / architectural style (not rounded doodle icons)
- Color: `#8a8a8a` default, `#da1313` when active/highlighted
- Consistent size within a single infographic

---

## Data Visualization

- **Process flows:** Horizontal arrow chain. Step boxes in `#121212` border `#333333`,
  step number in `#da1313`, label in `#f0f0f1`, sub-label in `#8a8a8a`.
  Feedback arcs (loops back) shown as a curved red arrow below the chain.
- **Matrices / grids:** Column headers in `#da1313`. Row labels left-aligned.
  "Primary" cells: filled `#da1313` dot or solid mark. "Secondary": outline mark. "None": empty.
- **Comparison tables:** Header row `#0f0f0f` with red top border. Alternating rows `#121212` / `#1a1a1a`.
- **Stat callouts:** Large number in `#da1313`, unit/label in `#f0f0f1` below.
- **Timelines:** Horizontal. Years/milestones in `#da1313`, event labels in `#f0f0f1`.
- **Bar/column charts:** Bars in `#da1313`, axis labels in `#8a8a8a`, no gridlines unless faint `#333333`.

## Infographic Formats Used

| Format | Best for |
|---|---|
| Process flow (horizontal steps) | Workflows, pipelines, how-it-works |
| Phase matrix (rows × columns) | Deliverable fit by project stage |
| Deliverable grid (2–3 col cards) | Service offerings overview |
| Notable projects list | Credentials / past work |
| Comparison (before/after, table) | AI transformation, options |
| Timeline | Company history, project duration |
| Stat block | Key numbers, pricing anchors |

---

## Voice in Infographic Copy

- Short. Direct. No adjectives that don't earn their place.
- Use the service taglines verbatim as infographic headlines.
- Data over claims: "€5,000" not "affordable"; "48 hours" not "fast turnaround".
- Active voice. "We fix the scene before it hits the farm." not "Scene preparation is included."
- Numbers draw the eye — put key figures in large type.

---

## File / Export Notes

- Preferred canvas: 1920×1080 (landscape) or 1080×1920 (portrait/social)
- Also useful: 1200×630 (Open Graph / LinkedIn preview)
- Export: PNG (lossless) or SVG where text must scale
- Fonts must be embedded or outlined before export
